# Desktop Weather Forecasting System

A Python-based desktop application that provides real-time weather updates using the OpenWeather API.

## Features
- Search weather by city name
- Displays temperature, weather condition, and emoji visualization
- GUI built using PyQt5
- Secure API key using environment variables (.env)
- Proper error handling for API and network failures

## Technologies Used
- Python
- PyQt5
- Requests
- OpenWeather API
- python-dotenv
- Git & GitHub

## Setup Instructions

1. Clone the repository:
   git clone https://github.com/Sumitsingh-dev1/desktop-weather-forecasting-system

2. Install dependencies:
   pip install -r requirements.txt

3. Create a .env file and add:
   WEATHER_API_KEY=your_api_key_here

4. Run the application:
   python main.py
